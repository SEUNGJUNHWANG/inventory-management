```
├── utils/                     # 유틸리티

│
└── 부품_대량등록_양식.xlsx      # 부품 대량 등록 양식 템플릿
```

## 2. 모듈별 역할 요약

| 모듈 | 파일 | 역할 |
|------|------|------|
| **core** | `constants.py` | 앱 버전, 색상 팔레트, 폰트, 메뉴 항목 등 전역 상수 |
| **core** | `config.py` | JSON 설정 파일 읽기/쓰기, 경로 관리 |
| **core** | `database.py` | 구글 시트 CRUD, 입출고 처리, BOM 기반 생산, 이력 관리 |
| **ui** | `widgets.py` | 재사용 가능한 UI 컴포넌트 (카드, 버튼, 테이블 등) |
| **ui/pages** | 각 페이지 | 화면별 독립 모듈 (렌더링, 이벤트 처리) |
| **utils** | `helpers.py` | 로깅, 스레드 실행, 날짜 포맷 등 공통 함수 |
| **utils** | `excel_utils.py` | 엑셀 양식 생성, 부품 파싱, BOM 임포트 |

## 3. 새 기능 추가 방법

### 3-1. 새 페이지 추가

1. `ui/pages/` 폴더에 새 파일 생성 (예: `barcode.py`)

```python
class BarcodePage:
    def __init__(self, app):
        self.app = app

    def render(self):
        # self.app.content_frame에 UI 구성
        # self.app.db로 데이터 접근
        pass
```

2. `core/constants.py`의 `MENU_ITEMS`에 메뉴 추가:

```python
MENU_ITEMS = [
    ...
    ("barcode", "📱 바코드 스캔"),
]
```

3. `main.py`의 `InventoryApp.__init__`에서 페이지 등록:

```python
from ui.pages.barcode import BarcodePage
self.pages["barcode"] = BarcodePage(self)
```

### 3-2. 데이터베이스 기능 추가

`core/database.py`의 `GoogleSheetsDB` 클래스에 새 메서드를 추가합니다.

### 3-3. 공통 위젯 추가

`ui/widgets.py`에 재사용 가능한 UI 컴포넌트를 추가합니다.

## 4. 버전 관리

`core/constants.py`에서 버전을 관리합니다:

```python
APP_VERSION = "2.0.0"  # Major.Minor.Patch
```

- **Major**: 큰 기능 변경 또는 호환성 변경
- **Minor**: 새 기능 추가
- **Patch**: 버그 수정

---

## 5. EXE 빌드 방법 (완성 후)

1. 명령 프롬프트에서 프로젝트 폴더로 이동
2. `build_exe.bat` 실행 (또는 아래 명령어 직접 실행)

```
pip install pyinstaller
pyinstaller --onefile --windowed --name "재고관리시스템" main.py
```

3. `dist/재고관리시스템.exe` 파일이 생성됩니다.
4. 이 EXE 파일과 `config.json`, JSON 키 파일만 있으면 다른 PC에서도 실행 가능합니다.

---

## 6. 주의사항

- `config.json`과 서비스 계정 JSON 키 파일은 **EXE와 같은 폴더**에 있어야 합니다.
- 구글 시트 API 무료 제한: 분당 읽기 60회, 쓰기 60회
- 대량 작업 시 자동으로 배치 처리하여 API 제한을 회피합니다.
