# 재고관리 시스템 v1.0 사용 가이드

안녕하세요. 10년 차 재고관리 전문가님의 업무 효율을 극대화하기 위해 개발된 맞춤형 재고관리 애플리케이션입니다.
본 애플리케이션은 **구글 시트**를 데이터베이스로 사용하여 실시간 동기화와 데이터 안정성을 보장하며, 요청하신 **모든 추가 기능(안전재고 알림, 이력 관리, 바코드 연동, 엑셀 리포트)**을 완벽하게 지원합니다. 또한, 기존에 보유하신 방대한 **BOM 엑셀 파일(약 3,000건의 부품 및 제품별 시트)을 한 번에 임포트하는 기능**을 추가로 구현했습니다.

모든 프로그램 설치와 파일 배포는 바탕화면의 `마누스 자재관리 어플개발` 폴더에 완료되었습니다.

---

## 1. 초기 설정 (구글 시트 연동)

프로그램을 처음 실행하면 구글 시트와 연동하는 설정 페이지가 나타납니다.

1.  **Google Cloud Console** (https://console.cloud.google.com/) 에 접속하여 새 프로젝트를 생성합니다.
2.  **Google Sheets API**와 **Google Drive API**를 활성화합니다.
3.  **서비스 계정**을 생성하고 **JSON 키 파일**을 다운로드하여 안전한 곳에 보관합니다.
4.  새로운 구글 스프레드시트를 생성하고, 방금 만든 서비스 계정의 이메일 주소를 '편집자'로 추가하여 공유합니다.
5.  앱의 설정 화면에서 다운로드한 **JSON 파일의 경로**를 선택하고, 생성한 **스프레드시트 URL**을 입력한 뒤 `연결 테스트 및 저장` 버튼을 누릅니다. (URL을 비워두면 앱이 자동으로 새 시트를 생성합니다.)

---

## 2. 핵심 기능 가이드

### 2.1. 기존 BOM 엑셀 파일 자동 임포트 (추가 구현 기능)
바탕화면에 있던 `테노바 2026년 BOM.xlsx` 파일을 분석한 결과, 약 3,000건의 부품 리스트와 다수의 제품 BOM 시트가 존재했습니다. 이를 일일이 등록하는 수고를 덜어드리기 위해 자동 임포트 기능을 추가했습니다.

*   **사용 방법:** 설정 화면 하단의 **[📂 BOM 엑셀 파일 임포트]** 버튼을 클릭하고 `테노바 2026년 BOM.xlsx` 파일을 선택하세요.
*   **작동 방식:** '부품리스트' 시트를 분석하여 부품 마스터를 생성하고, 나머지 시트들의 제품명과 소요량을 분석하여 제품 마스터 및 BOM 데이터를 구글 시트에 자동으로 구축합니다.

### 2.2. 제품 생산 및 BOM 기반 자동 출고
가장 중요하게 요청하신 핵심 기능입니다. 제품을 생산하면 레시피(BOM)에 따라 필요한 부품이 한 번에 출고 처리됩니다.

*   **사용 방법:** 사이드바의 **[🏭 제품 생산]** 메뉴로 이동합니다.
*   **작동 방식:** 제품 코드와 생산 수량을 입력하고 `생산 처리`를 누르면, 앱이 BOM 데이터를 확인하여 각 부품의 필요 수량을 계산합니다.
    *   **사전 검증:** 부품 재고가 단 1개라도 부족하면 생산이 진행되지 않고 어떤 부품이 몇 개 부족한지 알려줍니다.
    *   **자동 출고:** 재고가 충분하면 제품 재고는 증가하고, 연관된 모든 부품의 재고가 정확히 차감됩니다.

### 2.3. 바코드/QR 스캐너 연동
부품의 입고와 출고를 빠르고 정확하게 처리할 수 있습니다.

*   **사용 방법:** 시중의 USB 바코드 스캐너를 PC에 연결합니다. **[📥 부품 입고]** 또는 **[📤 부품 출고]** 화면에서 '품번' 입력창에 마우스 커서를 클릭(포커스)해 둔 상태로 바코드를 스캔합니다.
*   **작동 방식:** 스캔 즉시 품번이 입력되고, 앱이 실시간으로 해당 부품의 이름과 현재 재고를 화면에 표시해 줍니다. 수량만 입력 후 엔터를 치면 바로 처리가 완료됩니다.

### 2.4. 안전 재고 알림 기능
부품이 부족해져서 생산에 차질이 생기는 것을 방지합니다.

*   **작동 방식:** 부품을 등록할 때 '안전재고' 수량을 설정해 둡니다. 부품이 개별 출고되거나 제품 생산으로 인해 자동 출고되었을 때, 남은 재고가 안전재고 수량 이하로 떨어지면 대시보드에 즉시 **빨간색 경고(⚠️)**가 표시됩니다.

### 2.5. 입출고 이력 추적 및 원복(취소)
모든 재고 변동 내역은 투명하게 기록되며, 실수로 잘못 입력한 내역은 쉽게 되돌릴 수 있습니다.

*   **사용 방법:** **[📜 입출고 이력]** 메뉴에서 특정 기간의 모든 입고, 출고, 생산 내역을 확인할 수 있습니다.
*   **취소 기능:** 실수로 처리된 내역이 있다면 해당 줄을 마우스 우클릭하여 **'이 이력 취소(원복)'**를 선택하세요. 출고되었던 부품은 다시 재고가 늘어나고, 입고되었던 제품은 다시 재고가 줄어드는 등 완벽하게 원상복구됩니다.

### 2.6. 엑셀 리포트 다운로드
구글 시트의 데이터를 깔끔하게 포맷팅된 로컬 엑셀 파일로 저장하여 보고용으로 활용할 수 있습니다.

*   **사용 방법:** **[📊 리포트]** 메뉴에서 원하는 리포트 종류(부품 현황, 제품 현황, 입출고 이력, 전체 통합)를 선택하여 다운로드합니다.
*   **특징:** 안전재고가 부족한 부품은 엑셀 파일 내에서도 빨간색 배경으로 강조 표시되어 한눈에 파악할 수 있습니다.

---

## 3. 프로그램 실행 방법

1.  바탕화면의 `마누스 자재관리 어플개발` 폴더를 엽니다.
2.  **`재고관리시스템.bat`** 파일을 더블클릭하여 실행합니다.
3.  까만색 콘솔 창이 열리면서 곧이어 메인 재고관리 프로그램 화면이 나타납니다.

*(참고: 처음 실행 시 Python과 필수 라이브러리가 이미 설치되어 있으므로 바로 실행됩니다. 만약 다른 PC로 폴더를 옮겨서 사용하실 경우, 해당 PC에 Python을 설치한 뒤 `install_dependencies.bat` 파일을 한 번 실행해 주시면 됩니다.)*
